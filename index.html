¿Cómo usar este solver de cubo de Rubik?

Si quieres aprender cómo resolver el cubo de Rubik 3x3, esta herramienta es perfecta para ti. Nuestro simulador 3D te guía paso a paso.

  • Gira el cubo virtual arrastrando con el ratón o el dedo.
  • Copia los colores de tu cubo físico haciendo clic en los cuadraditos.
  • Al terminar, la aplicación te dará la lista de movimientos (U, R, L, F...) para armarlo.
